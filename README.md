

### Compilar una expresión

```bash
python compilador.py "result = a + b"
```

### Ejemplos

```bash
# Expresión simple
python compilador.py "result = a + b"

# Con múltiples operaciones
python compilador.py "result = a + b - c"

# Con paréntesis
python compilador.py "result = a + (b - c)"

# Expresión compleja
python compilador.py "result = a + b - c + (d - e) + f"
```
