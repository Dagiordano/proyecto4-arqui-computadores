# Compilador4 -  Pruebas

## Uso básico

```bash
python compilador4.py "result = <expresión>"
```

## Ejemplos de prueba

### Suma simple
```bash
python compilador4.py "result = a + b"
```

### Resta simple
```bash
python compilador4.py "result = a - b"
```

### Multiplicación
```bash
python compilador4.py "result = a * b"
```

### División
```bash
python compilador4.py "result = a / b"
```

### Módulo
```bash
python compilador4.py "result = a % b"
```

### Expresión con múltiples operaciones
```bash
python compilador4.py "result = a + b - c"
```

### Expresión con paréntesis
```bash
python compilador4.py "result = a + (b - c)"
```

### Expresión compleja
```bash
python compilador4.py "result = a * b + c / d"
```

### Expresión con precedencia
```bash
python compilador4.py "result = a + b * c"
```

### Expresión con signos unarios
```bash
python compilador4.py "result = -a + b"
```

### Expresión muy compleja
```bash
python compilador4.py "result = a + b * c - d / e + f % g"
```
